üõ†Ô∏è What is JSON Server?

JSON Server lets developers build a full fake REST API with minimal setup. This is especially useful for beginners to practice GET, POST, PATCH, and DELETE requests.

With just a simple JSON file, you can create endpoints that behave like a real server. This means you can test your frontend code, simulate data changes, and even collaborate with others before the actual backend is ready.

For example, you can create a db.json file with sample data, run JSON Server, and instantly have a working API at http://localhost:3000.
